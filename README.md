## What is VecFin?

VecFin brings together in one place the financial data from your digital wallets, bank accounts, and crypto brokers. The goal is to give users a complete picture of their net worth, track spending trends, and receive AI-powered personalized recommendations — all from a single platform.

---

## Features

| Module | Description |
|---|---|
| **Consolidated dashboard** | Unified summary of available liquidity and total asset value |
| **Platform linking** | Connect banks, digital wallets, and crypto brokers |
| **Trend analysis** | Spending projections and yield forecasts based on user history |
| **AI assistant** | Financial diagnostics and rebalancing suggestions generated by an LLM |
| **Community** | Thematic communities, posts, news feed, and user following |
| **Financial simulator** | Simulate fixed-term deposits and variable income instruments |

## Getting started

### Prerequisites

- [Go 1.22+](https://golang.org/dl/)
- [Node.js 18+](https://nodejs.org/)
- [Docker](https://docs.docker.com/get-docker/) and Docker Compose

### Backend

```bash
# 1. Clone the repository
git clone https://github.com/Masega360/vecfin.git
cd vecfin

# 2. Set up environment variables
cp .env.example .env
# Fill in the values in .env

# 3. Start with Docker (recommended)
docker compose up --build
```

The server will be available at `http://localhost:8080`.

To run without Docker:

```bash
docker compose up -d db   # database only
go run ./backend/cmd/api
```

### Mobile app

```bash
cd mobile
npm install
npx expo start
```

Scan the QR code with **Expo Go** on your phone ([Android](https://play.google.com/store/apps/details?id=host.exp.exponent) / [iOS](https://apps.apple.com/app/expo-go/id982107779)). Your phone and PC must be on the **same Wi-Fi network**.

> Not on the same network? Use `npx expo start --tunnel`

## Environment variables

| Variable | Description | Example |
|---|---|---|
| `DB_URL` | PostgreSQL connection string | `postgres://user:pass@localhost:5432/vecfin` |
| `JWT_SECRET` | Secret for token signing | `supersecret` |
| `AI_API_KEY` | LLM provider API key | `sk-...` |

| `PORT` | HTTP server port | `8080` |

## API — Main endpoints

### Implemented

```
POST   /auth/login              Sign in
POST   /auth/google             Sign in with Google
POST   /users                   Register a new user
GET    /profile                 Get authenticated user profile
GET    /users/:id               Get user by ID
PUT    /users/:id               Update profile
DELETE /users/:id               Delete account
```

### Planned

```
GET    /portfolio               Consolidated dashboard
GET    /portfolio/assets        Portfolio assets
POST   /portfolio/assets        Add asset manually

GET    /platforms               Linked platforms
POST   /platforms               Link a new platform
DELETE /platforms/:id           Unlink a platform

GET    /market/:symbol          Asset detail
GET    /market/search           Search assets

POST   /ai/diagnosis            Generate financial diagnosis
POST   /ai/rebalance            Get rebalancing suggestions
GET    /ai/history              AI interaction history

GET    /community               List communities
POST   /community               Create a community
POST   /community/:id/post      Post in a community
POST   /posts/:id/like          Like a post
POST   /users/:id/follow        Follow a user
```
